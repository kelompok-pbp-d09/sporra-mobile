## ğŸ“ **Deskripsi Aplikasi**

**Sporra** adalah aplikasi mobile yang menjadi wadah komunitas olahraga. Aplikasi ini memungkinkan pengguna untuk berdiskusi, mengikuti event olahraga, membaca berita, serta melihat aktivitas komunitas secara langsung melalui perangkat Android.

Tujuan utama aplikasi ini adalah menciptakan ruang interaktif bagi para penggemar berbagai cabang olahraga untuk saling bertukar informasi, membuat kegiatan, dan membangun komunitas yang aktif.

### **Fitur Utama Aplikasi**

Dengan Sporra Mobile, pengguna dapat:

* Melihat dan mengikuti berita terbaru dari dunia olahraga.
* Membuat, mengelola, dan bergabung ke berbagai event olahraga.
* Mengikuti diskusi melalui forum: posting, komentar, balasan, serta upvote.
* Mengelola profil dan melihat riwayat aktivitas.

Aplikasi dibangun secara modular dan menggunakan Django pada sisi backend.
